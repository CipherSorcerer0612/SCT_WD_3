# SCT_WD_3
QUIZ GAME APPLICATION Develop an interactive multiple-choice quiz game that displays questions, collects answers, and provides a score at the end. You can also, add different type of Questions like, Single or Multi Select, Fill in the Blanks or more.
 
 
 
 Features
- Multiple question types:
  - Single choice
  - Multi-select
  - Fill in the blanks
  - Order-based questions
- Timer-based quiz system
- Score and streak tracking
- Category-wise performance analysis
- Personalized feedback system

How to Run
1. Download or clone the repository
2. Open `index.html` in a browser
3. Start the quiz and view results
